# 🍽️ Food Ordering Behaviour Analysis

## 📌 Project Overview

The **Food Ordering Behaviour Analysis** project is an interactive data visualization project developed as part of the **AICTE Tableau Internship**. It analyzes customer food ordering patterns and consumer trends using Tableau dashboards and stories.

The project focuses on understanding customer preferences, popular cuisines, city-wise ordering trends, revenue patterns, delivery performance, and customer ratings using interactive Tableau visualizations.

A Flask-based web application is also developed to present the Tableau Dashboard and Story through a simple and user-friendly web interface.

---

## 🎯 Objectives

- Analyze customer food ordering behaviour and consumer trends.
- Identify popular cuisines and meal preferences.
- Compare order patterns across different cities.
- Analyze revenue generation and delivery performance.
- Evaluate customer ratings and satisfaction levels.
- Provide meaningful business insights using Tableau dashboards.
