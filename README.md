# Food Ordering Behaviour and Consumer Trends Analysis

## Project Overview
This project focuses on analyzing food ordering behaviour and consumer trends using Tableau. The dashboard provides insights into customer preferences, order patterns, cuisine popularity, city-wise performance, and delivery trends.

## Tools Used
- Tableau Public
- CSV Dataset
- Data Visualization

## Dataset Details
The dataset contains information about:
- Order ID
- Order Value
- Delivery Fee
- Cuisine
- Meal Type
- City
- Delivery Time

## Dashboard Features
- Key Performance Indicators (KPIs)
- Cuisine-wise analysis
- City-wise order analysis
- Meal type analysis
- Delivery time analysis
- Customer behaviour insights
